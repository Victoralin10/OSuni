# OSuni
Simulación de un sistema operativo como proyecto del curso ST324. Facultad de Ingeniería de Sistemas de la Universidad Nacional de Ingeniería. 

## Requerimientos
1. jdk1.8
2. gradle 3.5

## Compilar
gradle build

## Generar jar
gradle jar

## Ejecutar
1. crear el archivo `confi.ini` copiando `config.ini.example`
2. gradle run
